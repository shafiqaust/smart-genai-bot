Module: Introduction to Deep Learning

--- SECTION: Overview ---

In this module, we will discuss two examples of Deep Learning Neural Networks. Focus will be on setting up your machines to work with the code of the book "Generative Deep Learning" and discussing an example of a Multilayer Perceptron and an example of a Convolutional Network.

--- SECTION: Learning Objectives ---

Content

By the end of this module, you should be able to:

MLO 1: Understand the basics of neural networks
MLO 2: Be able to use Tensorflow and Keras
MLO 3: Understand the differences between types of layers
MLO 4: Be able to identify suitable architecture for a given application
