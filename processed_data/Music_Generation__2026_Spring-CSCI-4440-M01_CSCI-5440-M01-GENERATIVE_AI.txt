MODULE TITLE: Music Generation

SECTION: Overview
In this module, we will focus on generating music. We will learn how to create data and use the Transformer architecture to train a MuseGAN.

SECTION: Learning Objectives
By the end of this module, you should be able to:

MLO 1: Understand how we can treat music generation as a sequence prediction problem, so we can apply autoregressive models such as Transformers.

MLO 2: See how to parse and tokenize MIDI files using the music21 package to create a training set.

MLO 3: Learn how to use sine positional encoding.

MLO 4: Train a music-generating Transformer, with multiple inputs and outputs to handle note and duration.

MLO 5: Understand how to handle polyphonic music, including grid tokenization and event-based tokenization.

MLO 6: Train a MuseGAN model to generate multitrack music and use the MuseGAN to adjust different properties of the generated bars.
