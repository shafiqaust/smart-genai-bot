MODULE: Autoregressive Models

OVERVIEW
--------
In this module, we will study a different type of GenAI model, called Autoregressive Models. We will discuss the motivation behind this type of models, the typical architecture of such a model, and implementations for different applications.

LEARNING OBJECTIVES
-------------------
Section: Content

By the end of this module, you should be able to:

MLO 1: Understand why autoregressive models are well suited to generating sequential data such as text.

MLO 2: Understand the architectural design of recurrent neural networks (RNNs).

MLO 3: Build and train a long short-term memory network (LSTM) from scratch using Keras.

MLO 4: Understand how image data can be treated as a sequence of pixels.

MLO 4: Build a PixelCNN from scratch using Keras.
