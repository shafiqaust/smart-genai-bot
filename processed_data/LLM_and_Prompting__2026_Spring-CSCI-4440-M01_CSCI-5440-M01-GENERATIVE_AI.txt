SECTION: Overview

In this week and the following week, we will play with Large Language Model (LLM) and learn about prompting.

---

SECTION: Learning Objectives

Content

By the end of this module, you should be able to:

MLO 1: Understand the basic of LLM
MLO 2: Familiar with different prompting techniques
MLO 3: Understand the differences between different prompting techniques
MLO 4: Be able to identify suitable prompting techniques for a given task
