MODULE: World Model

--------------------------------------------------------------------------------
SECTION: Overview
--------------------------------------------------------------------------------
In this module, we will learn about reinforcement learning, world model, and the
architecture for creating an agent that can act optimally in a real-world domain.

--------------------------------------------------------------------------------
SECTION: Learning Objectives
--------------------------------------------------------------------------------
By the end of this module, you should be able to:

MLO 1: Understand the basics of reinforcement learning (RL).

MLO 2: Understand how generative modeling can be used within a world model approach.

MLO 3: Understand the architecture used in training an agent using reinforcement learning.

MLO 4: Learn the steps of the training process.
